# Brief-Credit-Maroc
npm
```bash
npm install
```

node
```bash
npm init
```
```bash
npm start
```
Installez Docker pour votre système d'exploitation.
Installez l'outil Docker Compose.
Exécutez docker-compose
```bash
docker-compose up -d --build
```
